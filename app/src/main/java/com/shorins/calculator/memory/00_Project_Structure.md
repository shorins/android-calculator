# Структура проекта

Проект представляет собой стандартное Android-приложение, написанное на Kotlin с использованием Jetpack Compose. Архитектура следует паттерну MVVM (Model-View-ViewModel).

## Основные файлы и директории

-   `app/build.gradle.kts`: Конфигурационный файл сборки для модуля `app`. Здесь объявляются зависимости, такие как Jetpack Compose, ViewModel и т.д.
-   `app/src/main/AndroidManifest.xml`: Манифест приложения, объявляющий компоненты, разрешения и основную информацию о приложении.

### Исходный код (`app/src/main/java/com/shorins/calculator`)

-   **`MainActivity.kt`**:
    -   Единственная Activity в приложении.
    -   Является точкой входа в UI.
    -   Инициализирует `CalculatorViewModel` и отображает `CalculatorScreen`.

-   **`ui/theme/`**:
    -   Стандартные файлы темы Jetpack Compose (`Color.kt`, `Theme.kt`, `Type.kt`), определяющие визуальный стиль приложения.

-   **`calculator/`**: Основной пакет, содержащий всю логику калькулятора.
    -   **`components/CalculatorButton.kt`**: Переиспользуемый UI-компонент для кнопок калькулятора.
    -   **`CalculatorScreen.kt`**: Главный Composable-файл, отвечающий за отображение всего UI калькулятора, включая логику переключения между портретным и ландшафтным режимами.
    -   **`CalculatorViewModel.kt`**: "Мозг" калькулятора. Обрабатывает все действия пользователя, выполняет вычисления и управляет состоянием.
    -   **`CalculatorState.kt`**: Простой data-класс, который хранит текущее состояние калькулятора (`number1`, `number2`, `operation`).
    -   **`CalculatorAction.kt`**: Sealed-класс, который определяет все возможные действия пользователя (нажатие цифры, операции, очистки и т.д.).
    -   **`CalculatorOperation.kt`**: Sealed-класс, определяющий доступные математические операции.
